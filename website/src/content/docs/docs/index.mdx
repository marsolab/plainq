**PlainQ** is a small, self-contained queue server written in Go. A single
binary gives you a gRPC API, a CLI, a terminal UI, and a built-in admin
dashboard (Houston) — backed by embedded SQLite for local and small
deployments, or PostgreSQL when you need to scale out.

<CardGrid stagger>
  <Card title="One binary" icon="rocket">
    Run `./plainq serve` and you have a queue. No broker fleet, no external
    dependencies.
  </Card>
  <Card title="gRPC + CLI + TUI" icon="laptop">
    The same surface, scripted or interactive — with `-json` everywhere for
    scripts and agents.
  </Card>
  <Card title="Pick your storage" icon="seti:db">
    Embedded SQLite by default (Litestream-friendly), or PostgreSQL for a shared
    backend.
  </Card>
  <Card title="Auth built in" icon="approve-check">
    JWT sessions, refresh tokens, RBAC, and OAuth/OIDC hooks ship with the
    server.
  </Card>
</CardGrid>
